# nlp_bayan — نموذج توليدي لغوي منطقي مبني على «البيان» (هيكل منفصل)

هذا المجلد يحوي نموذجاً توليدياً لغوياً Logic-first مبنياً على لغة «البيان»، معزولاً عن نواة اللغة لتجنّب تشتّت المفاهيم. النموذج يستفيد من الأنظمة التاريخية (قبل نضوج البيان) بعد ترحيل ما يلزم وتحديثه.

## الأهداف
- توليد نص عربي يلتزم بقيود منطقية ودلالية (كيانات، علاقات، احتمالات، شبكات سببية/دلالية).
- أنابيب Text→Logic→Inference→Realize لإنتاج جمل مفسَّرة ومُبرَّرة منطقياً.

## البنية
- core/
  - linguistic_equations.bayan: معادلات لغوية وقاموس كلمة→معادلة + قواعد استخراج.
  - probabilistic.bayan: حقائق احتمالية وأدوات التشكيك (maybe/likely/… بالعربي).
  - semantic_network.bayan: شبكة دلالية مبسطة (عُقد/علاقات واستعلامات).
  - operators.bayan: أنواع المشغّلات وقواعد إمكانية التطبيق.
  - entities.bayan: كيانات بسيطة (حالات/خصائص/أفعال/تفاعلات).
  - causal_network.bayan: واجهة شبكة سببية مبسطة.
  - generator_pipeline.bayan: واجهة التوليد المنطقي (الدالة generate_simple_ar).
- examples/
  - demo_generation.bayan: مثال تشغيل سريع.
- docs/
  - DESIGN_OVERVIEW_AR.md: تصميم وعلاقة الملفات القديمة بالجديدة.

## التشغيل السريع
من جذر المستودع:

```bash
python -m bayan nlp_bayan/examples/demo_generation.bayan
```

- يطبع جملة عربية مولّدة منطقياً.
- يمكنك فتح Web IDE وتشغيل المثال أيضاً.

## ملاحظات
- هذه نسخة أولية تعليمية بهدف تثبيت الهيكل وترسيم حدود الوحدات. لاحقاً يمكن تعميق الاستدلال والربط مع مكتبات ai/*.
