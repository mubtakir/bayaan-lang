# دليل استخدام قوالب الـ Prompts

## الهدف
توفير هيكل موحّد لتغذية النماذج بسياقات بشرية قابلة للتحويل إلى كيانات وأفعال وآثار في "بيّان".

## البنية العامة
- صيغة الملفات: JSON صارم (strict JSON)
- الحقول الأساسية:
  - `السياق_المنطقي`: وصف الكيانات وحالاتها/خصائصها
  - `الوصف_البشري`: سرد طبيعي للحدث
  - `المخرج_المطلوب`: { `الإجراء`, `القيمة` (0..1), `الأثر` }

## مثال مبسّط
```json
{
  "السياق_المنطقي": "كيان أحمد { حالات: طاقة=0.6 }",
  "الوصف_البشري": "أحمد متعب بعد العمل",
  "المخرج_المطلوب": {
    "الإجراء": "راحة",
    "القيمة": 0.9,
    "الأثر": "أحمد.طاقة = أحمد.طاقة + 0.3"
  }
}
```

## الاستخدام المقترح
1) اختر القالب المناسب (اجتماعي/فيزيائي)
2) املأ الحقول بنصوص واضحة وقيم رقمية
3) أرسل الناتج إلى محول/مقيّم لاحقًا (سيضاف في eval_framework)

## ملاحظات
- ترميز UTF-8
- القيم العددية يجب أن تكون أرقامًا حقيقية (وليس نصوصًا)
- افصل الآثار المتعددة بعلامة `;` داخل حقل `الأثر`
