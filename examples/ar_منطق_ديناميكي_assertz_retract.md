# منطق ديناميكي: assertz / retract

يوضح هذا المثال كيفيّة تحديث قاعدة المعرفة أثناء التشغيل بإضافة/حذف حقائق ثم الاستعلام عنها.

```bayan
hybrid {
    fact likes("أحمد", "قهوة").

    print("قبل الإضافة:")
    query likes("أحمد", ?ماذا).

    assertz(likes("أحمد", "شاي")).

    print("بعد الإضافة:")
    query likes("أحمد", ?ماذا).

    retract(likes("أحمد", "قهوة")).

    print("بعد الحذف:")
    query likes("أحمد", ?ماذا).
}
```
