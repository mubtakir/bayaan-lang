# SVG: رسم حر (gfx)

رسم قلب باستخدام قلم_تحرك/قلم_منحنى ثم تحويله إلى مسار.

```bayan
include "gfx/svg.bayan"

ق = قلم_جديد()
قلم_تحرك(ق, 240, 80)
# الفص الأيمن
قلم_منحنى(ق, 300, 20, 380, 60, 240, 200)
# الفص الأيسر
قلم_منحنى(ق, 100, 60, 180, 20, 240, 80)
قلم_إغلاق(ق)

داخل = قلم_إلى_مسار(ق, "#f9b", "#c33", 2)
صورة = لف_SVG(480, 260, داخل)
print(صورة)
```
