# نمذجة كيانات وأفعال (Thing-centric vs Action-centric)
نموذجان: قائم على الكيان وقائم على الفعل مع قاعدة تربطهما.

```bayan
hybrid {
    # تمثيل قائم على الكيان
    fact كائن("أحمد").
    fact جائع("أحمد", "مرتفع").

    # تمثيل قائم على الفعل
    fact أطعم("خالد", "أحمد").

    # قاعدة: الإطعام يُخفّض الجوع
    rule انخفض_الجوع(?شخص) :- أطعم(_, ?شخص).

    print("من الذي انخفض جوعه؟")
    query انخفض_الجوع(?من).
}
```
