# SVG: أشكال أساسية (gfx)

مستطيل ودائرة وخط ونص باستخدام دوال SVG.

```bayan
include "gfx/svg.bayan"

داخل = مستطيل(20, 20, 120, 80, "#fee", "#c33", 2)
داخل = داخل + دائرة(220, 80, 40, "#cdf", "#246", 2)
داخل = داخل + خط(20, 150, 360, 150, "#666", 1)
داخل = داخل + نص(30, 180, "مرحبًا SVG", 16, "#133")

صورة = لف_SVG(400, 220, داخل)
print(صورة)
```
