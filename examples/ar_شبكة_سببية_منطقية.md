# شبكة سببية منطقية (Causal Network)
مثال صغير يبين شبكة سببية وقاعدة انتقالية للاستنتاج.

```bayan
hybrid {
    # حقائق سببية — causal edges
    fact causes("التدخين", "السرطان").
    fact causes("السرطان", "الإرهاق").
    fact causes("القلق", "الأرق").

    # استنتاج سببي انتقالي
    rule يسبب_ضمنيًا(?س, ?ص) :- causes(?س, ?ص).
    rule يسبب_ضمنيًا(?س, ?ز) :- causes(?س, ?و), يسبب_ضمنيًا(?و, ?ز).

    print("ما الذي يسببه التدخين؟")
    query يسبب_ضمنيًا("التدخين", ?أثر).
}
```
